- git edge
- git status - mostrar status dos objetos
- mv - comando de mover objetos

- cd - comando para entrar em objetos. ex: "cd receitas/"
- cd .. - comando para voltar um nivel. ex: "cd receitas/receitas doces" ~ cd .. "cd receitas/".
